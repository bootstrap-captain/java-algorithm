## ✅[344. 反转字符串](https://leetcode.cn/problems/reverse-string/)

```java
public void reverseString(char[] s) {
    int first = 0;
    int second = s.length - 1;
    while (first < second) {
        char temp = s[first];
        s[first] = s[second];
        s[second] = temp;
        first++;
        second--;
    }
}
```
